Advanced Features

different kind of filters can be used, just by changing the url slightly:

  1. https://today.noi.bz.it#NOI to filter by organizer (the one under the event title), use '#' in the url
  2. https://today.noi.bz.it?fields=digital to filter by technology fields of the event, add the request paramter '?fields=<technology field>' to the url. You can also add multiple techology fields using ',' as a seperator
  3. https://today.noi.bz.it/?ofs={%22EventDescriptionEN%22:%22AlpineBits%22} advanced filter method to filter on any given json field(only for people who have some background)
